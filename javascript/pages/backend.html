Backend frameworks

NodeJS

Node.js representa el entorno de ejecución del código en JavaScript, que está construido sobre el motor JavaScript Chrome V8, que permite traducir llamadas en JavaScript a código máquina.

Express.js

Minimalist, flexible, simplifies building web servers and APIs, strong routing and middleware support, large community

NestJS

Combines OOP, FP, and FRP, encourages use of TypeScript, structured and organized, integrates well with other libraries
