Javascript Everywhere

El mundo actual de los sitios web difícilmente se puede imaginar sin el lenguaje JavaScript. JavaScript es lo que hace que las páginas web cobren vida, las mismas que navegamos a diario en nuestro navegador web. JavaScript fue creado en 1995 por Brendan Eich en Netscape como un lenguaje de secuencias de comandos para el navegador Netscape Navigator 2. Inicialmente, el lenguaje se llamaba LiveScript, pero debido a la popularidad en ese momento del lenguaje Java, LiveScript fue renombrado como JavaScript. Sin embargo, este punto aún puede causar cierta confusión: algunos desarrolladores principiantes creen que Java y JavaScript son casi el mismo lenguaje. No, son dos lenguajes completamente diferentes y solo comparten el nombre.

Inicialmente, JavaScript tenía capacidades bastante limitadas. Su objetivo era simplemente agregar un poco de comportamiento a las páginas web. Por ejemplo, manejar clics de botones en una página web o realizar otras acciones relacionadas principalmente con los elementos de control.

Sin embargo, el desarrollo del entorno web, la aparición de HTML5 y la tecnología Node.js han abierto horizontes mucho más amplios para JavaScript. Ahora, JavaScript se sigue utilizando principalmente para crear aplicaciones web del lado del cliente, pero ahora ofrece muchas más posibilidades.

JavaScript es un lenguaje interpretado. Esto significa que el código en JavaScript se ejecuta utilizando un intérprete. El intérprete recibe instrucciones en el lenguaje JavaScript, que están definidas en la página web, y las ejecuta (o interpreta).
